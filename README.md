# 9arrini

Mobile learning application designed for primary school students, providing structured lessons and interactive exercises across multiple subjects.

## Overview
9arrini aims to make learning more accessible and engaging by offering personalized educational content based on the student’s level. The application supports multiple languages and dynamically generates exercises and summaries.

## Features
- Grade-based learning (1st to 6th year)
- Subjects: Arabic, Mathematics, French, English
- Interactive exercises and summaries
- Dynamic content generation via external API
- Multilingual support

## Tech Stack
- React Native (Expo)
- JavaScript / TypeScript
- External API integration

## Project Structure
- `app/` – main screens and routing
- `components/` – reusable UI components
- `navigation/` – navigation logic
- `assets/` – images and static files
